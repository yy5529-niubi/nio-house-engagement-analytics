NIO House Engagement Analytics

This project develops a data-driven analytics framework to quantify customer engagement and evaluate the impact of community activities in NIO House.

Key Components
User segmentation based on behavioral KPIs
Activity impact analysis using quasi-experimental methods
KPI construction (visit frequency, retention, spend)
Tools
R (tidyverse, lubridate)
Outputs
Segmentation profiles
Activity performance ranking
Engagement uplift analysis
